Audio

There ae three supported audio formats: MP3, WAV, and OGG. The HTML <audio> element is used to play an audio file on a web page.

  • The controls attribute adds audio controls, like play, pause, and volume.
  • The <source> element allows you to specify alternative audio files which the browser may choose from. The browser will use the first recognized format.
  • The text between the <audio> and </audio> tags will only be displayed in browsers that do not support the <audio> element.
  • The appearance of the tag depends on the browser, but you can hide it, build your own interface and control the playback using JavaScript.
  • The audio tag can be used with its own attributes and the global atrtibutes.
Attributes Description
autoplay The audio will start automatically once the page loads
controls The control (play/pause, volume,...) buttons are displayed
loop The audio will restart after it ends
muted The audio is automatically muted
src The URL of the audio file

Example:

Code Text
<!DOCTYPE html>
<html>
<head>
<title>Some Title</title>
</head>
<body>

<audio controls autoplay loop muted src="Vespers%20on%20the%20Shore%20-%20The%20Mini%20Vandals.mp3" type="audio/mpeg">
This browser does not support audio files.
</audio>

Video

  • Use the controls attribute to add video controls, like play, pause, and volume.
  • Use the width and height attributes to indicate the width and height of the video.
  • Use the <source> element to specify alternative video files which the browser may choose from. The browser will use the first recognized format.
  • The text between the <video> and </video> will only be displayed in browsers that do not support the <video> element.

Example:

Code Text
<!DOCTYPE html>
<html>
<head>
<title>Some Title</title>
<body>

<video loop autoplay controls src="Jellyfish.mp4" style="width: 100%">
This browser does not support video files.
</video>
