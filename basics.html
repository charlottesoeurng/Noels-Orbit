Head Element

  • These are required in the head of an HTML document:
    • The <!DOCTYPE html> element defines HTML as the document type.
    • The <head> element is placed between the <html> tag and the <body> tag.
    • The <title> element is required and it defines the title of the document.
  • <!DOCTYPE html> doesn't need an end tag but <head>, <html>, <body>, and <title> do need end tags.

Example:

Code
<!DOCTYPE html>
<html>
<head>
<title>Some Title</title>
</head>
<body>

Attributes

HTML attributes provide additional information about HTML elements.

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"
  • The href attribute of <a> specifies the URL of the page the link goes to
  • The src attribute of <img> specifies the path to the image to be displayed
  • The width and height attribute of <img> provide size information for images
  • The alt attribute of <img> provides an alternate text for an image
  • The style attribute is used to add styles to an element, such as color, font, size, and more
  • The lang attribute of the <html> tag declare the language of the Web page
  • The title attribute defines some extra information about an element

Example: For the code <p style="color: green">, style="color: green" is an attribute.

Text Formatting

Formatting elements display different types of texts.

  • <b> displays bold texts.
  • <strong> displays important texts. The text inside this element is usually displayed in bold.
  • <i> displays italic texts
  • <em> displays emphasized texts. The text inside this element is usually displayed in italic.
  • <mark> displays marked texts. The text inside this element is usually highlighted.
  • <small> displays smaller texts.
  • <del> displays "deleted" texts. The text inside this element usually has a line through it.
  • <ins> displays "inserted" texts. The text inside this element is usually underlined.
  • <sub> displays subscript texts. The text inside this element is usually in a smaller font and below the normal line.
  • <sup> displays superscript texts. The text inside this element is usually in a smaller font and above the normal line.

There are many more colors not in the list above but you can get them by using HEX values, HSL values, RGB values, RGBA values, and HSLA values.

  • Hex color codes are values that tell the display how much of a color to show. A hex color code is a 6-symbol code. For example, #000000 is a hex color code.
  • A color can be specified using hue, saturation, and lightness. For example, hsl(0, 0%, 0%) is a HSL value.
    • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
    • Saturation is a percentage value. 0% means gray, 100% is the full color.
    • Lightness is also a percentage value. 0% is black, 100% is white.
  • RGB values indicate a color's red, green, and blue intensity. Each intensity value is on a scale of 0 to 255, or in a hexadecimal from 00 to FF. For example; rgb(0, 0, 0) is an RGB value.

Classes

  • The class attribute specifies one or more class names for an element
  • Classes are used by CSS and JavaScript to select and access specific elements
  • The class attribute can be used on any HTML element
  • The class name is case sensitive
  • Different HTML elements can point to the same class name

How it works is you write in the <head> tag how you want an element to display using a specific text and then you use the same text later in the body as an attribute for the code to work.



Links

  • Use the <a> element to define a link
  • Use the href attribute to define the link address
  • Use the target attribute to define where to open the linked document
  • HTML target values:
    • _blank: Opens the linked document in a new window or tab
    • _self (default): Opens the linked document in the same frame as it was clicked
    • _parent: Opens the linked document in the parent frame
    • _top: Opens the linked document in the full body of the window
    • framework: Opens the linked document in the name iframe
  • Use the <img> element (inside <a>) to use an image as a link
  • Use the mailto: scheme inside the href attribute to create a link that opens the user's email program
  • Use the id attribute (id=value) to define bookmarks in a page
  • Use the href attribute (href=#value) to link to the bookmark
